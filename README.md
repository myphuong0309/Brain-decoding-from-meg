### 1. Setup Environment

```bash
# Clone the repository
git clone <repository-url>

# Install dependencies
pip install -r requirements.txt
```

### 2. Data Preparation

Place your LibriBrain data in `assets/data/` with the following structure:
```
assets/data/
├── Sherlock1/
├── Sherlock2/
└── ...
```

### 3. Run Complete Pipeline

```bash
bash run.sh
```
